🔧 Project Title

Hardware Spillage and Sequence Missing Prevention

🚀 Overview

This project tackles two critical disruptions in modern manufacturing—hardware spillage and sequence missing—through innovative mechanical and IoT-based solutions aligned with Industry 4.0 goals.

🛠️ Core Challenges

  • Hardware Spillage: Loss of nuts, bolts, washers during handling and assembly.
  • Sequence Missing: Incorrect part selection or installation disrupting synchronized workflows.

🧠 Root Cause Analysis

Identified using Fishbone Diagram across:

  • Human error
  • Machine malfunction
  • Material inefficiencies
  • Poor workplace ergonomics
  • Process inconsistencies

🔩 Solution 1: Nut Cartridge-Feeder Device

Mechanism: Rack-and-pinion + magnetic nut alignment

  • Automated nut handling
  • Prevents spillage
  • Improves ergonomics and cycle time
Nut Cartridge vs Pneumatic Feeder
Feature Cartridge Device Pneumatic Feeder
Accuracy ✅ Magnetic precision ⚠️ Air-flow dependent
Maintenance ✅ Low ⚠️ High
Energy Use ✅ Passive ⚠️ High (Compressed Air)

🔄 Solution 2: Sequence Control

🌱 Spring-Based System

  • Weight-based deflection guides selection
  • Radium screen reveals correct part
  • Ideal for semi-automated stations

🌐 IoT Smart Compartment

  • Load cells + RGB LEDs
  • Live alerts: green (ready), blue (next), red (error)
  • Arduino-powered with supervisor notifications

📈 Results & Performance

  • ✅ 100% sequence error reduction
  • ✅ Improved workflow and pick accuracy
  • ✅ Real-time tracking and ERP compatibility
